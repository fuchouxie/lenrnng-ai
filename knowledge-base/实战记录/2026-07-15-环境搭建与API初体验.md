## 解决的问题

### Windows Python 环境问题

**症状**：`python --version` 无输出

**原因**：Windows 应用执行别名中的 Python 占位程序排在 PATH 最前面，找到了假 python

**解决**：Win+I → 搜索"应用执行别名" → 关闭 Python 相关开关 → 重开终端

**真正的 Python 位置**：`C:\Users\admin\AppData\Local\Programs\Python\Python311\python.exe`

### load_dotenv 找不到 .env

**症状**：`KeyError: 'DEEPSEEK_API_KEY'`

**原因**：VS Code 运行目录和 .env 文件不在同一位置

**解决**：`load_dotenv()` 改为 `load_dotenv(r"D:\project\lenrnng-ai\ai-playground\.env")`
